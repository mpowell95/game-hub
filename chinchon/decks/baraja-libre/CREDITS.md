# Card artwork — Baraja Española (default deck)

The Spanish-suited card faces in this folder are from an open, freely-licensed
"Baraja Española" vector set.

- **Source:** https://github.com/gjenkins20/spanish-playing-cards-svg
- **Original artwork:** Basquetteur / Germarquezm, via Wikimedia Commons —
  https://commons.wikimedia.org/wiki/File:Baraja_espa%C3%B1ola.svg
- **License:** Creative Commons Attribution-ShareAlike 3.0 Unported (CC BY-SA 3.0) —
  https://creativecommons.org/licenses/by-sa/3.0/
- **Changes made:** files renamed to the game's `<suit>-<rank>` convention and
  optimized for the web (precision reduction / re-encoding).

Per CC BY-SA 3.0, these card images — and any modifications to them — remain
licensed under CC BY-SA 3.0, with attribution as above. The game's own source
code is separate and is not affected by bundling these images (a collection, not
a derivative of the artwork).
