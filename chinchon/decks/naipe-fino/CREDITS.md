# Card artwork — Naipe Fino (gold medallion deck)

A custom Spanish-suited deck built around a single supplied gold coin image.

- **Oros (coins):** the project-supplied gold "NAIPE FINO" medallion (`coin_clean.png`),
  composited onto card faces at the traditional Spanish pip positions.
- **Copas / Espadas / Bastos:** embossed gold suit medallions (cup / sword / club)
  generated as vector art for this project, styled to match the coin.
- **Figures (Sota / Caballo / Rey):** the suit medallion plus a gold crest
  (shield / horseshoe / crown) and a name banner — generated for this project.
- **Card back:** navy-and-gold seal built around the coin — generated for this project.

All faces are rasterized to WebP at 400 px wide, flattened onto an opaque card
background. The generated (non-coin) art was produced specifically for this game;
the supplied coin image is used with the project owner's permission. This deck is
independent of the default `baraja-libre` deck and its CC BY-SA 3.0 license.
