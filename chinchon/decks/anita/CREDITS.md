# Card artwork — Anita (a personal custom deck)

"Anita" is a fully custom deck built for this project. It began as a light skin over the
default **Baraja Española** deck, then grew card by card until it now provides **all 48
faces + a back**, so the Española fallback is no longer used at runtime.

- **Oros 1–9:** a project-supplied gold coin bearing "Ana" as an embossed portrait
  medallion (`coin_clean.png`), composited onto a white card in the traditional Spanish
  pip layout.
- **Copas / Bastos / Espadas 1–9:** original vector emblems drawn for this deck — a beer
  mug (copas), a golf driver (bastos), and a pickleball paddle (espadas), reflecting the
  group's real pastimes.
- **Court cards (Sota / Caballo / Rey, ranks 10–12, all four suits):** 12 AI-illustrated
  *baraja española* court cards depicting the project's own people (Ana and Matt) in
  period costume themed to each suit (coins, beer, golf, pickleball). Generated with an
  external AI image tool from the project owner's own reference photos; the framing and
  game code are original.
- **Back (`back.webp`):** a project-supplied personal photo of Ana, background removed and
  composited into an original card-back design (deep green field, gold engine-turned
  pattern and border, the coin as a centre medallion, 180°-rotational symmetry).
- **Betty win/lose art (`betty-win.webp`, `betty-loss.webp`):** project-supplied personal
  photos, background removed, shown on the end-of-match screen when this deck is active.

All faces are original or owner-supplied; this deck no longer reuses the CC BY-SA 3.0
Española art. (The Española deck's own attribution remains in `../baraja-libre/CREDITS.md`.)
